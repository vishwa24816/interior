# **App Name**: Frame & Hue

## Core Features:

- Animated Project Portfolio: Showcase interior design projects using engaging frame motion effects for transitions and reveals.
- Services Section: Detail the interior design services offered, enhanced with subtle animations to highlight key aspects.
- AI-Powered Design Tool Suggestion: Recommends design choices to users with similar projects in their portfolio based on client preferences using AI.
- About Us Page: Provide an overview of the company, its history, and its values, incorporating smooth frame motion for a visually appealing narrative.
- Contact Form: Enable potential clients to easily reach out, integrated with a simple and intuitive design for effective lead generation.

## Style Guidelines:

- Primary color: Soft Green (#BBD1B1) to evoke feelings of trust.
- Background color: Warm Gray (#EDEDED), providing a sophisticated backdrop that highlights content without distraction.
- Accent color: Cream (#F5F5DC), used sparingly to draw attention to key interactive elements.
- Font pairing: 'Playfair' (serif) for headings and 'PT Sans' (sans-serif) for body, creating an elegant and readable text experience.
- Use a clean, grid-based layout to showcase interior design projects effectively.
- Implement subtle frame motion effects for smooth transitions and content reveals, enhancing user engagement.
